       - Replace `private initialized = false;` with `private initPromise: Promise<void> | null = null;`
       - Change `initialize()` method body to:
         ```typescript
         async initialize(): Promise<void> {
           if (!this.initPromise) {
             this.initPromise = this.doInitialize();
           }
           return this.initPromise;
         }

         private async doInitialize(): Promise<void> {
           logger.info({ maxBrowsers: this.maxBrowsers }, 'Initializing browser pool');
           for (let i = 0; i < this.maxBrowsers; i++) {
             const browser = await chromium.launch({ args: BROWSER_LAUNCH_ARGS });
             this.browsers.push(browser);
           }
           logger.info({ count: this.browsers.length }, 'Browser pool ready');
         }
         ```
       - Update `getBrowser()` check: replace `if (!this.initialized` with `if (!this.initPromise`
       - Update `close()`: set `this.initPromise = null` instead of `this.initialized = false`
